# 🚀 FutureMe AI

## 🌟 About

FutureMe AI is a career planning platform that helps students and aspiring professionals discover the right learning path based on their skills, interests, and career goals.

The current version focuses on providing an intuitive user interface and structured workflow for collecting career-related information. Future updates will integrate AI to generate personalized learning roadmaps, recommend projects, identify skill gaps, and provide career guidance.

---

## 🎯 Problem Statement

Many students struggle to decide:
- Which skills to learn next
- Which projects to build
- How to prepare for their dream career
- Where to begin their learning journey

FutureMe AI aims to simplify this process by creating a personalized career roadmap for every user.

## 💡 Workflow

1. User enters career details.
2. User selects experience level.
3. System collects user information.
4. *(Upcoming)* AI analyzes the profile.
5. *(Upcoming)* Personalized roadmap is generated.
6. User follows the suggested learning path.
